# 🛒 Electronics Shop App – React Native

A simple e-commerce mobile application built with React Native.  
The app allows users to browse electronic products, add items to a cart, and view total price and quantity in real time.

---

## 🎯 Project Purpose

This project was built to practice core React Native concepts including:

- Building a multi-screen mobile application
- Managing global state using Context API
- Implementing shopping cart logic
- Handling dynamic calculations (total price, quantity)
- Working with navigation between screens

---

## ✨ Features

- 🛍️ Browse list of electronic products  
- ➕ Add products to shopping cart  
- 🧺 View cart with selected items  
- 🔢 Update product quantity in cart  
- 💰 Automatic total price calculation  
- 🧭 Navigation between Home and Cart screens  

---

## 🛠️ Tech Stack

## 🚀 Getting Started

### Prerequisites
- Node.js  
- npm or yarn  
- Expo CLI or React Native CLI  

---

### Installation

Clone the repository:
git clone https://github.com/your-username/electronics-shop-app.git

Navigate to project folder:

cd electronics-shop-app

Install dependencies:

npm install
Run the App

Using Expo:

expo start

Or React Native CLI:

npm run android
# or
npm run ios
